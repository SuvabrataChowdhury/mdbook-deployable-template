# Contributing to mdbook-deployable-template

Thank you for considering contributing to this project! We welcome contributions from everyone, regardless of experience level.

This template is designed to be **inclusive and accessible**, and we want to maintain that spirit in our community. Whether you're fixing a typo, improving documentation, or adding a feature, your contribution matters.

---

## 🤝 Ways to Contribute

You don't need to be a developer to help! Here are ways you can contribute:

### 📝 **Documentation & Writing**
- Improve clarity in the README or guides
- Fix typos or grammar
- Add examples or tutorials
- Expand the FAQ
- Improve inline code comments

### 🐛 **Bug Reports**
- Found something broken? [Open an issue](https://github.com/SuvabrataChowdhury/mdbook-deployable-template/issues)
- Include what went wrong, what you expected, and how to reproduce it

### 💡 **Feature Requests & Ideas**
- Have a suggestion? [Start a discussion](https://github.com/SuvabrataChowdhury/mdbook-deployable-template/discussions)
- We love hearing how people want to use this

### 🎯 **Code Contributions**
- Fix bugs
- Add small features (check [open issues](https://github.com/SuvabrataChowdhury/mdbook-deployable-template/issues) first)
- Improve automation (GitHub Actions, scripts, etc.)

### 🌟 **Community**
- Share your success story in [Show and tell](https://github.com/SuvabrataChowdhury/mdbook-deployable-template/discussions/categories/show-and-tell)
- Answer questions in [Discussions](https://github.com/SuvabrataChowdhury/mdbook-deployable-template/discussions)
- Help others who are getting started

---

## 🚀 How to Contribute Code

### 1. **Check Existing Issues**
Before starting, look through [open issues](https://github.com/SuvabrataChowdhury/mdbook-deployable-template/issues) and [discussions](https://github.com/SuvabrataChowdhury/mdbook-deployable-template/discussions) to see if someone is already working on it.

Issues marked with **`good first issue`** are great for newcomers!

### 2. **Fork the Repository**
Click the "Fork" button to create your own copy.

```bash
git clone https://github.com/YOUR-USERNAME/mdbook-deployable-template.git
cd mdbook-deployable-template
