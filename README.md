# 📖 mdbook-deployable-template

**Publish a professional, auto-deployed static site with just Markdown—no coding required.**

Use this template to create and publish blog-style websites, technical documentation, study guides, or any readable content. Write in Markdown, push to GitHub, and your site is live. That's it.

---

## ✨ Why Use This?

- **Zero configuration** — Everything is pre-configured. Just write content.
- **No coding required** — If you can write Markdown and use Git, you can publish.
- **Auto-deploy** — Push to GitHub → Site deploys automatically via GitHub Actions.
- **Free hosting** — Hosted on GitHub Pages at no cost.
- **Professional look** — Clean, readable design out of the box.

**Perfect for:**
- Students writing notes or tutorials
- Authors publishing short books or technical guides
- Educators sharing course material
- Technical writers documenting projects
- Anyone who wants a clean website without building it

---

## 🚀 Quick Start

### Get up and running in 3 steps:

1. **Use this template** — Click **"Use this template"** button at the top of this repository
2. **Enable GitHub Pages** — Go to Settings → Pages → Set source to "GitHub Actions"
3. **Start writing** — Edit files in `src/` and push to GitHub

✅ Your site will be live in 1-2 minutes!

## ❓ Quick FAQ

**Q: Do I need to know how to code?**
No. You only need to write Markdown and use Git/GitHub.

**Q: Can I preview locally?**
Yes! Use `mdbook serve` to see live changes as you edit.

**Q: How do I add pages?**
Create Markdown files in `src/` and link them in `src/SUMMARY.md`.

**Q: Can multiple people work on this?**
Yes! Collaborate using Git branches and pull requests.

**Q: What if I have questions?**
