# Usage

Ensure that you have `mdbook` cli installed. If not install it with,
```bash
brew install mdbook # For macOS
```

## Steps
1. Clone this repo.
2. Create a branch and start writing your book.
    - While writing preview your changes with command
        ```bash
        mdbook serve
        ```
3. Once you are satisfied with your changes push changes to master. A job will automatically kick in and upload the results to github-pages.
