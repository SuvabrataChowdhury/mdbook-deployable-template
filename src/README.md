# Introduction

Ever wondered about publishing a static book-site but not a real fan of coding? Well just write your book with [mdbook](https://rust-lang.github.io/mdBook/format/index.html) and the repo will automatically take care of deployment for you.

## Purpose

This project's purpose is to ease out publishing book sites. The only knowledge the author needs for publishing their book-sites are,
1. **Markdown files**, as all the contents are to be written in this format.
2. **Git/ GitHub**, just for version control of the contents.

The setup automatically takes care of deploying the generated build to [github pages](https://docs.github.com/en/pages/getting-started-with-github-pages/what-is-github-pages).
